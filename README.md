# Project: Frontend Integration Testing

This project facilitates comprehensive integration testing for frontend applications, combining BackstopJS for visual regression testing, Selenium for triggering frontend button controls, and Docker for running web projects in isolated environments. Additionally, the project supports Cucumber for behavior-driven testing, allowing tests to be segmented into various scenarios based on user behavior.

## Technology Stack

- Java
- Maven
- Docker
- Spring Boot

## Dependencies

Ensure the following tools are installed in your local environment:

- Java Development Kit (JDK)
- Maven
- Docker
